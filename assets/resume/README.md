# Resume Folder

Place your resume PDF file in this folder with one of the following names:

- `Bhuvanesh Resume.pdf` (preferred - with space)
- `Bhuvanesh_Resume.pdf` (with underscore)
- `resume.pdf`
- `Resume.pdf`
- `CV.pdf`
- `cv.pdf`

The portfolio will automatically detect and enable the download button when you add your resume file.
